# Secret Links

パスワードを入力するとリンク集を表示する、GitHub Pages用の簡易ロック付きリンク集サイトです。

## 機能

- パスワード入力画面
- SHA-256ハッシュによる簡易チェック
- ログイン後にリンク集を表示
- ログアウト機能
- sessionStorageによるログイン状態の一時保存
- スマホ対応デザイン

## 使用技術

- HTML
- CSS
- JavaScript
- Web Crypto API
- GitHub Pages

## 注意事項

このサイトはGitHub Pages上で動く簡易ロックです。

本格的な認証機能ではありません。

個人情報、APIキー、本物のパスワード、秘密情報などは置かないでください。
